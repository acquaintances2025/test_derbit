Описание проекта:

Проект написан на Python 3.12 c использованием асинхронного фреймворка FastAPI 0.128.0
В проекте использована монолитная архитектура с полным разделением сущностей на исполнительные процессы,
конфигурации приложения, вспомогательные элементы таких как работа с базами данных, проксирующий клиент, логгер,
а так же основное ядро приложения.
Получение курсов и сохранность их в базу данных выведена в отдельную подзадачу которая не взаимосвязана
с основным кодом приложения. 

Развертывание:

Приложение сделано на базе двух контейнеров docker (база данных postgres и основного приложения),
таблица для базы данных разворачивается автоматически при запуске приложения. Все конфиги настроены и 
тянутся автоматически.
Для развертывания проекта хватает ввести docker-compose up -d
